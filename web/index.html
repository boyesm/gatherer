How it Works

A web scraper constantly scrapes the web for new emails associated with web pages. If it finds a hit, it is saved to a MongoDB run in the cloud. This website queries the database based on the domain of the email (the part after the @) Then, it shows these emails based on the number of times it shows up on the web, and potentially other metrics as well.
